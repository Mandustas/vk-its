# ВК ИТС — Сайт компании

Современный сайт для компании ВК ИТС, разработчика и интегратора интеллектуальных систем видеоаналитики и распознавания.

## Технологии

- **Vite 5** — быстрая сборка проекта
- **React 18** — UI библиотека
- **TypeScript 5** — типизация
- **React Router DOM 6** — навигация
- **SCSS** — стилизация

## Структура проекта

```
vk-its/
├── public/              # Статические файлы
│   ├── favicon.svg
│   ├── llms.txt        # Карта для LLM
│   ├── humans.txt      # Информация о команде
│   ├── robots.txt      # Правила для поисковиков
│   └── sitemap.xml     # Sitemap
├── src/
│   ├── components/      # Переиспользуемые компоненты
│   │   ├── Header.tsx
│   │   ├── Footer.tsx
│   │   └── SchemaOrg.tsx
│   ├── pages/           # Страницы приложения (14)
│   ├── data/            # Данные (продукты, решения, кейсы)
│   ├── styles/          # Глобальные стили
│   │   ├── variables.scss
│   │   └── index.scss
│   ├── App.tsx          # Корневой компонент
│   └── main.tsx         # Точка входа
└── index.html
```

## Запуск проекта

### Установка зависимостей
```bash
npm install
```

### Запуск dev-сервера
```bash
npm run dev
```

Сервер запустится на порту 3000: http://localhost:3000

### Сборка для продакшена
```bash
npm run build
```

### Предпросмотр продакшен сборки
```bash
npm run preview
```

## Страницы сайта

### Основные
- **Главная** (/) — Hero-блок, продукты, решения, кейсы, новости, статистика, CTA
- **Продукты** (/products) — Каталог продуктов с детальными страницами
- **Сравнение** (/compare) — Сравнение продуктов с таблицей характеристик
- **Решения** (/solutions) — Отраслевые решения (транспорт, промышленность, умный город)
- **Кейсы** (/cases) — Реализованные проекты с фильтрацией по отраслям
- **О компании** (/about) — Информация, цифры, документы, новости

### База знаний
- **Глоссарий** (/knowledge) — 25+ терминов по видеоаналитике и ИТС
- **Ответы на вопросы** (/answers) — 6 развёрнутых статей по популярным вопросам
- **Полезные материалы** (/resources) — 12 лид-магнитов (чек-листы, калькуляторы, шаблоны)

### Информация
- **Документы** (/documents) — 15 документов (сертификаты, ТД, презентации)
- **Поддержка** (/support) — FAQ (4 категории, 15+ вопросов), контакты
- **Поиск** (/search) — Поиск по продуктам, решениям, кейсам, страницам
- **Контакты** (/contacts) — Форма обратной связи, карта, реквизиты

## Дизайн-система

### Цветовая палитра
- **Основной**: `#2563eb` (синий)
- **Акцент**: `#06b6d4` (циан)
- **Вторичный**: `#0f172a` (тёмный)
- **Успех**: `#10b981` (зелёный)
- **Градиент**: `linear-gradient(135deg, #2563eb 0%, #06b6d4 100%)`

### Компоненты
- Кнопки (primary, secondary, outline)
- Карточки (card, product-card, solution-card, case-card)
- Сетки (grid--2, grid--3, grid--4)
- Секции (section, section--gray)
- Аккордеоны (FAQ)
- Выпадающие меню (dropdown)
- Модальные окна

### Типографика
- Шрифт: Inter / system-ui
- Иерархия: h1 (3rem) → h2 (2.25rem) → h3 (1.5rem)
- Межстрочное расстояние: 1.6
